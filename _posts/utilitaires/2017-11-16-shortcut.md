# Linux
* Resize window : super + r
* Move window : super + alt + space

# VS Code
* New window : ctrl + shift + n
* New tab : ctrl + n
* Open Recent : ctrl + shift + o
* Show terminal : ctrl + shift + ^
* Rewrap text : alt + q
